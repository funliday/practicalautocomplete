# otocomplete

Autocomplete 就是使用者在輸入文字的時候，server 可以即時回應資料給 client 呈現對應的內容

## Autocomplete 原理

* 依照字母排序
* 依照使用頻率排序
* Read times >>>>> write times
* 時間複雜度
